# Prototype roomCAMERA

[python project](https://github.com/HackerShackOfficial/Smart-Security-Camera/)

집안에 설치하여 감시카메라 목적으로 설치하였습니다.   
* 안면을 탐지하여 다양한 기능을 구현할 수 있는 프로젝트 입니다.

## 기능

### 객체 감지

* 얼굴감지
* 상반신감지
* 몸체감지
* 콜라보 감지

### 저장기능

사진 저장(객체 탐지,사용자 지정), 영상 저장(객체 탐지,사용자 지정)  

### 사용자에게 메일 전송 

비디오 전송, 사진 전송  

## Running
```
$ python3 main.py
```
